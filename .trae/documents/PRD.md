# 猫猫对战小游戏 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目名称
**猫猫对战 (Cat Clash Claws)**

### 1.2 项目类型
基于 Web 的 2D 回合制对战小游戏

### 1.3 核心价值
为用户提供一个可爱、温馨的两只猫咪对战体验，融合角色个性和视觉美感。

### 1.4 目标用户
- 喜欢可爱风格的休闲游戏玩家
- 猫咪爱好者
- 寻找轻松对战游戏的用户

## 3. 游戏玩法设计

### 3.1 游戏类型
回合制 1v1 对战游戏

### 3.2 游戏流程
1. **开始界面**：显示游戏标题和"开始战斗"按钮
2. **角色选择**：展示两只猫猫供玩家选择
3. **战斗准备**：显示战斗场景和角色位置
4. **回合战斗**：交替进行攻击和防御操作
5. **胜负判定**：HP归零的一方失败
6. **结果界面**：显示胜利者和"再来一局"按钮

### 3.3 操作方式
**玩家1控制（阿黑）：**
- `W` / `↑`：上移
- `S` / `↓`：下移
- `A` / `←`：左移
- `D` / `→`：右移
- `J`：普通攻击
- `K`：防御
- `L`：技能攻击（消耗能量）

**玩家2控制（小白）：**
- `↑`：`W`
- `↓`：`S`
- `←`：`A`
- `→`：`D`
- `Numpad 1`：普通攻击
- `Numpad 2`：防御
- `Numpad 3`：技能攻击

### 3.4 战斗机制

#### 3.4.1 攻击系统
- **普通攻击**：造成攻击力80%-100%的伤害，有冷却时间(0.5秒)
- **技能攻击**：
  - 阿黑：利爪突袭，造成攻击力150%伤害
  - 小白：绒毛护盾，提升防御力100%，持续1回合
- **伤害计算**：实际伤害 = 攻击方攻击力 - 防守方防御力（最小为1）

#### 3.4.2 防御系统
- 防御状态：减少50%受到的伤害
- 防御持续：当前回合
- 防御动画：猫咪蜷缩，闪烁微光

#### 3.4.3 能量系统
- 最大能量：100点
- 普通攻击：+10点能量
- 技能攻击：消耗30点能量
- 能量自动恢复：每回合+5点

### 3.5 胜负条件
- 当一方HP降至0或以下时，另一方获胜
- 平局：同时HP归零

## 4. 视觉设计方案

### 4.1 整体风格
**唯美可爱风格 (Kawaii Aesthetic)**
- 柔和的色调
- 圆润的形状
- 可爱的表情
- 流畅的动画

### 4.2 配色方案
**主色调：**
- 天空蓝：`#E8F4FD` (背景)
- 樱花粉：`#FFB6C1` (强调色)
- 奶油白：`#FFF8E7` (UI背景)
- 薄荷绿：`#98D8C8` (成功/治疗)

**角色配色：**
- 阿黑：黑色 `#2C2C2C`，黄色眼睛 `#FFD700`
- 小白：白色 `#FFFFFF`，淡黄色眼睛 `#F5DEB3`

**UI配色：**
- 血量条：红色 `#FF6B6B` → 黄色 `#FFE66D` → 绿色 `#4ECDC4`
- 能量条：蓝色 `#74B9FF`
- 按钮：`#FF9FF3` (主)，`#54A0FF` (次)

### 4.3 字体选择
- **标题字体**：`Ma Shan Zheng` 或 `ZCOOL KuaiLe` (可爱中文手写体)
- **正文字体**：`Noto Sans SC` 或系统默认无衬线
- **英文装饰**：`Fredoka One` (圆润可爱)

### 4.4 场景设计

#### 4.4.1 主战斗场景
**天空花园背景：**
- 淡蓝渐变天空
- 漂浮的云朵
- 远处的粉色樱花树
- 软绵绵的草地
- 飘落的花瓣粒子效果

**战斗台：**
- 圆形的软垫平台
- 边缘装饰有小星星
- 轻微的阴影和层次感

## 5. 动画设计

### 5.1 角色动画

#### 5.1.1 待机动画
- **阿黑**：尾巴缓慢摇摆，耳朵偶尔转动，眨眼效果
- **小白**：身体轻微呼吸起伏，耳朵抖动，胡须微微颤动

#### 5.1.2 移动动画
- 轻盈的滑动效果
- 身体略微前倾
- 尾巴跟随移动方向
- 移动轨迹留下淡淡的脚印

#### 5.1.3 攻击动画
- **阿黑**：
  - 普通攻击：快速伸出爪子，爪痕特效
  - 技能攻击：身体旋转，黑色残影效果
- **小白**：
  - 普通攻击：伸出小爪子，轻微跳起
  - 技能攻击：全身毛发炸开，发出柔和光芒

#### 5.1.4 防御动画
- 猫咪蜷缩成球
- 身体周围出现透明光圈
- 轻微的摇晃效果

#### 5.1.5 受伤动画
- 身体后仰
- 出现星星/音符等晕眩效果
- 血量条闪烁

#### 5.1.6 胜利/失败动画
- **胜利**：开心跳跃，爪爪挥舞
- **失败**：趴下，耳朵下垂

### 5.2 UI动画

#### 5.2.1 血量条动画
- 血量变化时平滑过渡
- 血量低时闪烁警告
- 受伤时红色脉冲效果

#### 5.2.2 按钮动画
- 悬停时放大1.05倍
- 点击时缩小0.95倍
- 背景色渐变过渡

#### 5.2.3 伤害数字动画
- 从攻击位置弹出
- 向上飘动并逐渐消失
- 暴击时显示金色并放大

### 5.3 粒子效果
- 移动时的可爱脚印
- 攻击时的星星/爪痕
- 胜利时的彩带/爱心
- 场景中的飘落花瓣

## 8. 技术实现方案

### 8.1 技术栈
- **前端框架**：纯 HTML5 + CSS3 + JavaScript
- **绘图方式**：Canvas 2D
- **动画**：CSS Animations + requestAnimationFrame
- **音效**：Web Audio API（可选）

### 8.2 文件结构
```
/
├── index.html          # 主入口
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── main.js         # 游戏主逻辑
│   ├── cat.js          # 猫猫角色类
│   ├── battle.js       # 战斗系统
│   └── ui.js           # UI管理
└── assets/
    └── (内联SVG/CSS绘制)
```

### 8.3 核心类设计
- `Game`：游戏主控制器
- `Cat`：猫猫角色基类
- `BattleSystem`：战斗逻辑处理器
- `UIManager`：界面管理
