# 猫猫对战小游戏 - 产品需求文档 (PRD) v2.0

## 1. 项目概述

### 1.1 项目名称
**猫猫对战 (Cat Clash Claws) v2.0**

### 1.2 项目类型
基于 Web 的 2D 实时/回合制混合对战小游戏

### 1.3 核心价值
为用户提供一个可爱、温馨且富有深度的两只猫咪对战体验，融合角色个性、视觉美感和策略深度。

### 1.4 目标用户
- 喜欢可爱风格的休闲游戏玩家
- 猫咪爱好者
- 寻找轻松但有深度的对战游戏的用户
- 移动端和桌面端用户

## 3. 游戏玩法设计 v2.0

### 3.1 游戏类型
实时对战 + 多回合制

### 3.2 游戏流程
1. **开始界面**：显示游戏标题、模式选择、难度选择
2. **战斗准备**：显示战斗场景和角色位置
3. **回合战斗**：
   - 自由移动和攻击
   - 使用暗器和技能
   - 拾取补血物品
4. **回合结束**：一方HP归零
5. **多回合制**：三局两胜
6. **结果界面**：显示最终胜者和比分

### 3.3 操作方式 v2.0

**玩家1控制（黑茶）：**
- `W` `A` `S` `D`：移动
- `J`：普通攻击（近战）
- `K`：防御
- `L`：技能攻击（暗影爆发）
- `U`：发射手里剑（远程）
- **连按2次方向键**：快速移动（双倍速度）
- **连按3次方向键**：冲撞（冲到屏幕边界，6-15伤害，60秒冷却）

**玩家2控制（茉莉）：**
- `↑` `←` `↓` `→`：移动
- `Numpad 1`：普通攻击
- `Numpad 2`：防御
- `Numpad 3`：技能攻击（神圣光芒）
- `Numpad 0`：发射星星（远程）

### 3.4 战斗机制 v2.0

#### 3.4.1 攻击系统
- **普通攻击**：
  - 近战范围：220像素
  - 超出范围自动发射暗器
  - 伤害：攻击力×(0.8-1.0)
  - 冷却时间：25帧
  - 命中回复：+3能量

- **远程暗器**：
  - 自动追踪目标
  - 伤害：3-6随机
  - 飞行速度：8像素/帧
  - 命中后消失

- **技能攻击**：
  - 消耗50能量
  - 冷却180帧
  - 效果因角色而异

#### 3.4.2 冲撞系统（新增）
- **触发**：快速连按3次同一方向键
- **效果**：
  - 冲向面向方向的屏幕边界
  - 0.5秒完成冲撞
  - 碰撞到敌人造成6-15随机伤害
  - 将敌人撞飞到屏幕边缘
- **冷却**：60秒
- **提示**：冷却中会显示"没力气了..."

#### 3.4.3 防御系统
- 防御状态：减少50%受到的伤害
- 防御持续：0.6秒
- 防御动画：猫咪蜷缩，紫色光圈

#### 3.4.4 能量系统
- 最大能量：100点
- 普通攻击：+3点能量
- 暗器发射：-10点能量
- 技能攻击：-50点能量
- 能量自动恢复：每帧+0.15点

#### 3.4.5 特殊机制
- **暴击**：20%概率，伤害×1.35
- **背刺**：从背后攻击，伤害×1.5
- **闪避**：15%概率闪避攻击
- **连击**：连续攻击同一目标，伤害递增（每连+10%）
- **物理碰撞**：两只猫不能重叠，自动弹开

#### 3.4.6 对话系统（新增）
- **触发条件**：血量低于70%
- **触发间隔**：5-40秒随机
- **黑茶对话**："喵~"、"喵！"、"喵呜~"、"喵喵！"
- **茉莉对话**："你把我打疼了！"、"快停下黑茶！"、"你个傻猫！"等
- **音效**：黑茶男声、茉莉女声

### 3.5 补血物品系统（新增）

#### 3.5.1 物品类型
- **猫粮** 🐱：恢复3-7 HP，概率35%
- **鸡肉冻干** 🍗：恢复5-10 HP，概率28%，有咬碎音效
- **罐头** 🥫：恢复8-15 HP，概率22%，有吸水音效
- **鱼干** 🐟：恢复4-9 HP，概率15%

#### 3.5.2 物品生成
- 每5秒随机生成一个
- 从屏幕上方落下
- 部分会停留在地面
- 停留10秒后消失

### 3.6 多回合系统（新增）
- **回合数**：最多3回合
- **胜利条件**：先赢2回合
- **比分显示**：实时显示当前比分
- **回合切换**：自动重置HP和位置

### 3.7 胜负条件
- 回合胜利：一方HP降至0
- 比赛胜利：先赢2回合
- 平局：同时HP归零（重新开始该回合）

## 5. AI系统 v2.0

### 5.1 难度等级

**简单 (Easy)：**
- 反应延迟：800ms
- 决策间隔：60帧
- 随机行为：50%概率
- 暗器使用：较少

**普通 (Medium)：**
- 反应延迟：400ms
- 决策间隔：40帧
- 随机行为：30%
- 暗器使用：一般

**困难 (Hard)：**
- 反应延迟：150ms
- 决策间隔：20帧
- 随机行为：10%
- 暗器使用：频繁
- 优先攻击低血量目标

### 5.2 AI行为逻辑

1. **血量<30%时**：优先防御
2. **距离<220时**：普通攻击
3. **距离>280时**：发射暗器
4. **距离>300时**：接近目标
5. **能量充足且冷却完毕**：使用技能
6. **目标血量<30%且困难模式**：疯狂追击

## 7. 技术实现方案 v2.0

### 7.1 技术栈
- **前端框架**：纯 HTML5 + CSS3 + JavaScript ES6+
- **绘图方式**：Canvas 2D
- **动画**：requestAnimationFrame
- **音效**：Web Audio API（程序化合成）
- **字体**：Google Fonts - Ma Shan Zheng, Noto Sans SC

### 7.2 核心文件结构
```
/
├── index.html              # 主入口
├── README.md              # 项目说明
├── css/
│   └── style.css         # 样式文件
├── js/
│   ├── game.js           # 游戏主控制器
│   ├── cat.js            # 猫猫角色类
│   ├── battle.js         # 战斗系统
│   ├── ai.js            # AI系统
│   ├── ui.js            # UI管理
│   ├── audio.js         # 音效系统
│   └── utils.js         # 工具函数
├── images/               # 角色图片
└── .trae/
    └── documents/       # 开发文档
```
