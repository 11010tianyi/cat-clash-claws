## 🎮 游戏介绍

**猫猫对战** 是一款基于 Web 的 2D 猫咪对战小游戏。玩家可以控制两只可爱的猫咪角色（黑茶和茉莉）进行对战，体验丰富的战斗系统、炫酷的技能特效和可爱的角色互动。

### ✨ 新版本 2.0 特性

- 🔥 **多回合制战斗** - 支持三局两胜，更有挑战性
- ⚡ **快速移动系统** - 连按两下方向键加速，连按三下冲撞
- 🗡️ **暗器系统** - 远程投掷攻击，各具特色
- 💬 **智能对话系统** - 血量低时会随机说话
- 🎵 **丰富音效** - 攻击、技能、对话、补血都有独特音效
- 🤖 **增强AI** - AI 会使用暗器和技能
- 💫 **炫酷技能特效** - 暗影爆发 vs 神圣光芒

## 🕹️ 操作指南

### 玩家1 - 黑茶（黑猫）

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| `J` | 普通攻击 |
| `K` | 防御 |
| `L` | 技能（消耗50能量） |
| `U` | 发射暗器 |

### 玩家2 - 茉莉（白猫）

| 按键 | 功能 |
|------|------|
| `↑` `←` `↓` `→` | 移动 |
| `Numpad 1` | 普通攻击 |
| `Numpad 2` | 防御 |
| `Numpad 3` | 技能（消耗50能量） |
| `Numpad 0` | 发射暗器 |

### 🎯 特殊操作

- **连按2次方向键**：快速移动（双倍速度）
- **连按3次方向键**：冲撞！冲向屏幕边界，造成6-15伤害（60秒冷却）
- **长按方向键**：持续移动

## 🎮 游戏机制

### ⚔️ 战斗系统

- **普通攻击**：近战，造成基础伤害
- **远程暗器**：可发射手里剑/星星进行远程攻击
- **技能攻击**：消耗50能量，释放强力技能
- **防御姿态**：减少50%受到的伤害

### 🎯 特殊机制

- **暴击**：20%概率造成1.35倍伤害
- **背刺**：从背后攻击造成1.5倍伤害
- **连击**：连续攻击同一个目标伤害递增
- **闪避**：15%概率闪避攻击

### 🏆 胜负规则

- 三局两胜制
- 任意一方血量归零则该回合结束
- 先赢两回合者获胜
