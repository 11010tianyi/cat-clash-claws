## 🎮 游戏介绍

**猫猫对战** 是一款基于 Web 的 2D 猫咪对战小游戏。玩家可以控制两只可爱的猫咪角色（黑茶和茉莉）进行对战，体验丰富的战斗系统、炫酷的技能特效和可爱的角色互动。

### ✨ 版本 2.3 新特性

- 🧠 **智能AI系统** - 增强版AI决策系统，反应更快更精准
- 🎯 **智能食物争夺** - AI会根据分数选择最优食物和避免负分食物
- ⚡ **AI难度细化** - 3个难度等级各具特色
- 🐀 **反噬鼠机制** - 打中反噬鼠会受到10点伤害
- 📊 **实时难度显示** - 小游戏模式也显示AI难度
- 📷 **真实场景图库** - 支持照片作为无限平铺背景
- 🎭 **打耗子小游戏** - 4种耗子（灰鼠/飞鼠/隐身鼠/反噬鼠）
- 🏆 **积分系统** - 打耗子得积分，累计排名
- 💫 **超现实手绘** - 樱花庭院、竹海秘境、峡谷遗迹
- 🔥 **多回合制战斗** - 支持三局两胜
- ⚡ **快速移动系统** - 连按两下方向键加速，连按三下冲撞
- 🗡️ **暗器系统** - 远程投掷攻击，各具特色
- 💬 **智能对话系统** - 血量低时会随机说话
- 🎵 **丰富音效** - 攻击、技能、对话、补血都有独特音效

## 🕹️ 操作指南

### 玩家1 - 黑茶（黑猫）

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| `J` | 普通攻击 |
| `K` | 防御 |
| `L` | 技能（消耗50能量） |
| `U` | 发射暗器 |

### 玩家2 - 茉莉（白猫）

| 按键 | 功能 |
|------|------|
| `↑` `←` `↓` `→` | 移动 |
| `Numpad 1` | 普通攻击 |
| `Numpad 2` | 防御 |
| `Numpad 3` | 技能（消耗50能量） |
| `Numpad 0` | 发射暗器 |

### 🎯 特殊操作

- **连按2次方向键**：快速移动（双倍速度）
- **连按3次方向键**：冲撞！冲向屏幕边界，造成6-15伤害（60秒冷却）
- **长按方向键**：持续移动

## 🎮 小游戏模式

### 🍖 抢食物

**规则**：
- 90秒内抢夺食物得分
- 猫草：黑茶-2分，茉莉+2分
- 塑料袋：黑茶+2分，茉莉-2分
- 普通食物：双方各+1分
- 得分高者获胜！

**AI难度**：
- **简单**：速度慢，经常犹豫，随性移动
- **挑战**：速度适中，策略稳定
- **大师**：速度快，精准判断，智能选择食物

### 🐀 打耗子

4种耗子等你来抓：

| 耗子类型 | 特征 | 速度 | 血量 | 积分 | 特殊能力 |
|----------|------|------|------|------|----------|
| 🐀 灰鼠 | 普通耗子 | 中速 | 1 | 1 | - |
| 🐁 飞鼠 | 快速耗子 | 高速 | 1 | 2 | 移动极快 |
| 👻 隐身鼠 | 半透明耗子 | 中速 | 1 | 2 | 隐身 |
| 💢 反噬鼠 | 危险耗子 | 中速 | 2 | 3 | 被打时反噬10点伤害 |

**游戏时长**：90 秒
**积分系统**：每只耗子 1-3 分
**注意事项**：小心反噬鼠！打中会受到伤害！

## 🧠 AI系统

### 🤖 智能决策

AI会根据以下因素做出决策：
- **距离判断**：近战攻击 vs 远程暗器
- **血量分析**：低血量时优先防御
- **能量管理**：能量足够才使用技能
- **战术优先级**：根据战况动态调整

### 📊 难度等级

| 难度 | 反应时间 | 决策延迟 | 特点 |
|------|---------|---------|------|
| 简单 | 800ms | 60帧 | 经常犹豫，随性行动 |
| 挑战 | 400ms | 40帧 | 稳定发挥，策略平衡 |
| 大师 | 85ms | 11帧 | 反应极快，精准打击 |

### 🎯 AI行为模式

**简单模式**：
- 速度较慢（48%）
- 经常犹豫不决（20%概率）
- 随机移动（28%概率）

**挑战模式**：
- 速度适中（56%）
- 偶尔犹豫（16%概率）
- 策略性移动（14%概率）

**大师模式**：
- 速度快（90%）
- 极少犹豫（4%概率）
- 高度专注（6%随机）

## 🎮 游戏机制

### ⚔️ 战斗系统

- **普通攻击**：近战，造成基础伤害
- **远程暗器**：可发射手里剑/星星进行远程攻击
- **技能攻击**：消耗50能量，释放强力技能
- **防御姿态**：减少50%受到的伤害

### 🎯 特殊机制

- **暴击**：20%概率造成1.35倍伤害
- **背刺**：从背后攻击造成1.5倍伤害
- **连击**：连续攻击同一个目标伤害递增
- **闪避**：15%概率闪避攻击

### 🏆 胜负规则

- 三局两胜制
- 任意一方血量归零则该回合结束
- 先赢两回合者获胜

## 🖼️ 添加自定义照片场景

1. 将照片文件（.jpg/.png/.webp）放入 `images/scenes/photos/` 文件夹
2. 在 `images/scenes/photos/manifest.json` 中添加文件名
3. 游戏会自动识别并加载你的场景！

```json
{
  "files": [
    "sakura.jpg",
    "bamboo.jpg",
    "your-custom-photo.jpg"
  ]
}
```

## 🎯 技巧与策略

### 🐱 对战技巧

- **利用地形**：大场地适合拉开距离用暗器
- **能量管理**：关键时刻使用技能扭转战局
- **闪避时机**：预判对手攻击，及时闪避
- **冲撞使用**：60秒冷却，优先击退敌人

### 🍖 抢食物技巧

- **猫草陷阱**：引诱对手去吃猫草扣分
- **速度优势**：大师AI会精准抢食物，你需要更快！
- **负分规避**：塑料袋对茉莉是负分，引诱她去吃

### 🐀 打耗子技巧

- **飞鼠要快**：飞鼠速度很快，要预判位置
- **反噬鼠危险**：打中会受到10点伤害，考虑击杀顺序
- **隐身鼠**：半透明，容易漏掉
- **积分最大化**：优先打高分耗子（飞鼠/隐身鼠/反噬鼠）
